博客图床

2020.02.28

博客图片一直存储在 Github 上，国内访问一直很慢，因此需要寻找合适的图床。在看了国内七牛、又拍云、贴图库等存储库之后，终因为备案、限量等各种原因，在快要放弃的时候发现了 jsDelivr 这个项目。

该项目旨在帮助大家使用 Github 作为静态文件的存储库，提供 CDN 访问支持，以下为本博客使用该项目更换图床的记录。

Git 提交静态文件
