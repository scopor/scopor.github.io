Maven 跳过某个子 Module 的构建

2020.03.18

当项目中出现某个稳定的模块，不需要每次编译的时候重复去构建它，我们可以使用如下的命令去跳过它

mvn -pl '!父Module/子Module' clean install

假如你使用的是 IDEA 插件 Maven Helper，那需要去掉引号

mvn -pl !父Module/子Module clean install

如果有多个子 Module 需要跳过，直接逗号隔开即可

mvn -pl '!父Module/子Module1,!父Module/子Module2' clean install

注: -pl 参数需要 Maven 3.2.1 以上版本
