如何杀掉 Windows 下占用某端口的进程

2020.12.08

我们在 Windows 上开发时偶尔会遇到指定端口被进程占用的情况，可以使用下面的方法查询并杀掉该进程。

查看占用 80 端口的进程

netstat -ano | findstr '80'

注意分辨，可能占用 8080 端口的进程也被查询出来了，确认找到你要杀掉的进程号，执行以下命令

taskkill -PID 进程号 -F

如果你习惯使用 Git Bash, 那么可以使用下面的命令一键查杀

netstat -ano | awk '{print $5,$2}' | grep ':80$' | awk '{print $1}' | xargs tskill
