Fork 库同步远程新增分支

2020.05.29

在开发过程中，经常会遇到自己的 Fork 库需要同步远程仓库新增的分支的情况，下面我们就来介绍下，如何进行同步。

获取远程新增分支信息

git fetch remote remote_new_branch_name:remote_new_branch_name

注: 请注意将 remote 和 remote_new_branch_name 替换称自己的远程仓库及分支名称

切换本地分支为远程新增的分支

git checkout remote_new_branch_name

推送本地分支到自己的 Fork 库

git push origin remote

关联本地新分支与远程新分支

git branch --set-upstream-to=origin/remote_new_branch_name remote_new_branch_name

至此，Fork 库同步远程新增分支完成。
