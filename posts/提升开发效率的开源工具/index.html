提升开发效率的开源工具

2020.08.12

本文记录一些开发过程中常用的开源工具，持续更新中。

Arthas

Arthas 是 Alibaba 开源的 Java 诊断工具。支持JDK 6+，支持Linux/Mac/Windows，采用命令行交互模式，同时提供丰富的 Tab 自动补全功能，进一步方便进行问题的定位和诊断。

SOAR

SOAR (SQL Optimizer And Rewriter) 是一个对 SQL 进行优化和改写的自动化工具。 由小米人工智能与云平台的数据库团队开发与维护。

Code To Flow

利用 Typora 笔记的代码种类 mermaid，自动将代码生成流程图。
