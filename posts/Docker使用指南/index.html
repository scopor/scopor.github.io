Docker 使用指南

2020.03.06

查看镜像列表

docker images
docker images -a

搜索镜像

docker search java

下载镜像

docker pull java

运行容器

docker run 镜像ID

停止容器

docker stop 容器ID
# 批量停止容器
docker stop `docker ps -a -q`

重启容器

docker restart 容器ID

启动容器

docker start 容器ID

删除容器

docker rm -f 容器ID
# 批量删除容器
docker rm `docker ps -a -q`

查看容器列表

# 运行中的容器
docker ps
# 所有容器，包括停止的
docker ps -a

查看容器日志

docker logs 容器ID

进入容器

docker exec -it 容器ID /bin/bash
docker exec -it 容器ID /bin/sh

查看容器配置

docker inspect 容器ID

查看容器占用资源的情况

# 默认每秒一次
docker stats
# 只查询当前的状态
docker stats --no-stream

镜像导出

docker save 镜像ID > path/file

镜像导入

docker load < path/file
