MacOS 系统 Go 的安装与配置

2020.08.21

本文主要学习在 MacOS 系统中，如何安装与配置 Go 语言的开发环境。

安装 Go

首先我们先下载 Go 安装包，MacOS 请选择 pkg，双击安装即可。安装完成后，我们来使用

go version 测试一下，可以看到我们安装的 Go 是 1.15 的版本。

➜  ~ go version
go version go1.15 darwin/amd64

使用命令 which go 可以查看 go 被安装到什么位置。

➜  ~ which go
/usr/local/go/bin/go

环境配置

安装完以后，Go 自动为我们配置好了环境变量信息，我们可以使用 go env 命令查看。

配置项较多，这里我只列一下关键信息。

➜  ~ go env
GOPATH="/Users/xxx/go"
GOROOT="/usr/local/go"

此时在用户 xxx 下面创建文件目录 go 即可，此外我们还需查看 PATH 信息中是否存在 GOPATH。

➜  ~ echo $PATH
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/go/bin

开发环境

因长期使用IDEA的关系，所以我选择继续使用 Jetbrains 出品的 GoLand 作为 Go 项目的开发工具，这里我使用的版本是 2020.1.2，安装好之后会自动识别环境变量GOPATH 和 GOROOT，我们可以验证下 Go 语言介绍 中的 Web 程序。

然后运行 go run main.go，访问浏览器 localhost:8080，可以看到 Hi, Lemon.

至此我们完成了 Go 开发环境的搭建。
