JVM 指令与工具

2020.03.18

查看 Java 进程列表: jps

jps

查看 gc 信息: jstat

jstat -gcutil PID 毫秒数


jstat -gc PID 毫秒数
jstat -gcnew PID
jstat -gcold PID

查看线程信息: jstack

jstack PID

查看内存状态: jmap

# 导出到文件中，需要使用其它工具打开  
jmap -dump:file=file PID
# 直接在控制台查看  
jmap -heap PID

查看进程扩展参数信息：jinfo

jinfo PID

内存溢出时自动导出内存信息

-XX:+HeapDumpOnOutOfMemoryError

打印 GC 日志, 使用 GCeasy 在线工具导入可分析

-XX:+PrintGCDetails -XX:+PrintGCTimeStamps -XX:+PrintGCDateStamps -Xloggc:./gc.log
