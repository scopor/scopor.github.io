Hexo 搭建博客

2020.06.16

Hexo 简介

快速、简洁且高效的博客框架，Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Hexo 安装

  • 前提是安装 Node && Git
  • 安装 hexo-cli 客户端命令
npm install -g hexo-cli
  • 创建项目
创建工程

hexo init blog

安装依赖

cd blog
npm install

启动服务，访问http://localhost:4000

hexo server

Hexo 主题安装

Hexo 提供了丰富的主题可供选择，请移步至此查看：Hexo主题，本博客选择了 maupassant 主题。

  • 安装主题
cd blog
git clone https://github.com/tufu9441/maupassant-hexo.git themes/maupassant
  • 安装相关插件
主题依赖的插件
npm install hexo-renderer-jade@0.3.0 --save
npm install hexo-renderer-sass --save
订阅插件和网站地图插件
npm install hexo-generator-feed --save
npm install hexo-generator-sitemap --save

其中安装 hexo-renderer-sass 时如遇到错误，可使用代理或者切换至 cnpm 执行。

  • 修改默认主题为 maupassant

编辑 Hexo 目录下的 _config.yml 文件，将 theme 的值改为 maupassant ，重新启动 hexo server 。

Hexo 全局配置

查看项目结构，会发现根目录下和主题目录下分别有 _config.xml 配置文件，其中根目录下的为全局配置文件.

title: Sanoon                   ## 博客主标题
subtitle: 行到水穷，坐看云起       ## 博客副标题
keywords:                       ## 关键字，可多个，逗号隔开，同时修改themes/maupassant/layout/_partial/head.jade，加上meta(name='keywords', content=config.keywords)，重启
description:                    ## 网站描述信息
author: Sanoon                  ## 作者
language: zh-CN                 ## 语言
timezone: Asia/Shanghai         ## 时区

url: https://sanoon.me/         ## 主页
root: /                         ## 根目录
permalink: /:category/:title/   ## 文章 url 路径，默认为时间格式 + 标题，此处修改为分类+标题
permalink_defaults:             ## 永久链接

source_dir: source              ## 文章所在目录

public_dir: public              ## 打包生成的静态文件所在目录
tag_dir: tags                   ## 标签文件夹，在 public 下
archive_dir: archives           ## 静态页面文章所在目录，在 public 下
category_dir: categories        ## 分类文件夹，在 public 下

highlight:                      ## 高亮配置
    enable: true
    line_number: true
    auto_detect: true

index_generator:                ## 页面设置
    path: ''
    per_page: 10                ## 每页显示的文章数
    order_by: -date             ## 排序，默认按照 时间倒序

plugin:                         ## 安装的插件
- hexo-generator-feed           ## RSS 订阅插件
- hexo-generator-sitemap        ## 网站地图插件

sitemap:
    path: sitemap.xml

feed:                           ## RSS 订阅配置
type: atom
path: atom.xml
limit: 20

theme: maupassant               ## 主题

写文章

在根目下的 source 文件夹结构如下

_posts                          ## 该目录为文章主目录
about                           ## 关于我的介绍，创建 index.md 文件。
catelories                      ## 存放分类的目录

其中每篇文章的开头需配置如下，内容写在该区域下方，语法遵循 Markdown，文件为 md 文件

使用 --- 进行闭合
---
title: Hexo 搭建博客             ## 文章名称
tags:                           ## 文章标签，可多个
- Hexo
categories: Tech                ## 文章分类，可以多个，按照[分类1，分类2]配置，默认支持 1 个，配置多个时，按照分类展示时，第 2 个分类会放在第 1 个url后面，形如：/catelogory1/catelogory2
date: 2017-11-29 10:36:21       ## 文章创作时间，排序的关键字段
keywords:                       ## 文章页的关键字
description:                    ## 摘要描述，文章在列表显示时候的内容区域
---

### 内容区域

设置关键字

在根目下添加 keywords 和 description 配置:

keywords: 首页的关键字
description: 网站描述

在主题下面的 _partial/hdea.jad 文件中找到 keywords 和 description,替换成如下代码:

if page.keywords
        meta(name='keywords', content=page.keywords)
    else
        meta(name='keywords', content=config.keywords)
    if page.description
        meta(name='description', content=page.description)
    else
        meta(name='description', content=config.description)

然后在写文章的的时候添加 keywords 和 description 配置

域名及云服务器购买

可在GoDaddy、gandi、阿里云万网购买域名，其中.cn的域名需要备案。

主机可选择 阿里云、DigitalOcean、腾讯云、Linode、亚马逊云、谷歌云等，此处不再赘述。

部署

Hexo 可以选择部署到 Github Pages 上，本站直接将生成的 public 目录下的文件上传到云服务器进行发布。

配置好服务器后，安装 Nginx 服务，具体参见配置。
